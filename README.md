# Tip Calculator (Tip Time)

## 1. Название проекта

Tip Calculator — это мобильное приложение для расчета чаевых, разработанное на платформе Android. Приложение позволяет пользователям быстро и удобно вычислить сумму чаевых на основе суммы счета и выбранного процента, с возможностью округления результата.

## 2. Цель работы

Изучение концепции состояния (state) в Android-приложениях и практическое применение Jetpack Compose для создания реактивного пользовательского интерфейса. В ходе работы было разработано приложение-калькулятор чаевых, а также освоены основы автоматизированного тестирования Android-приложений (локальные и инструментальные тесты).

## 3. Функциональность

- Ввод суммы счета с помощью цифровой клавиатуры
- Ввод процента чаевых (настраиваемое значение)
- Автоматический расчет суммы чаевых в реальном времени
- Опция округления суммы чаевых вверх (Round up)
- Отображение результата в формате валюты (с учетом локали устройства)
- Адаптивный интерфейс с поддержкой портретной и альбомной ориентации (вертикальная прокрутка)
- Визуальные подсказки: иконки для полей ввода (деньги и процент)
- Удобная навигация по полям ввода с помощью клавиатуры (Next → Done)

## 4. Технологии

- Язык программирования: Kotlin
- Фреймворк для UI: Jetpack Compose
- Дизайн-система: Material Design 3
- Управление состоянием: MutableState, remember, State Hoisting
- Тестирование:
    - Unit Tests (локальные тесты)
    - Compose UI Testing (инструментальные тесты)
- Инструменты: Android Studio, Resource Manager, GitHub
